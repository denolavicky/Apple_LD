# 🍎 Apple Nigeria Website Clone

A structured recreation of the Apple Nigeria website built using HTML and SCSS, with a proper team workflow using Git and GitHub.

---

## 📌 Project Overview

This project recreates key sections of the Apple Nigeria website:

- Landing Page  
- iPhone Page  

The goal is to practice clean structure, modular SCSS architecture, and real-world collaboration workflow.

---

## 🚀 Tech Stack

- HTML5  
- SCSS (compiled to CSS)  
- Git & GitHub  

---

## 👥 Team Structure

Each developer is responsible for both the HTML and SCSS styling of their assigned section.

- 🚀 Hero Section  
- 📱 Product / Lineup Section  
- 🧾 product Section 2 (Grid section/get to know the iphones section)  
- 🎨 Footer section 
- 😎 Navbar + Assigned Section (significant other on the iphone page) 

⚠️ Every developer works only on their assigned section.

---

## 🌿 Branching Strategy (Important)

All developers must work on separate branches.

🚫 No direct pushes to `main`.

### Branch Naming Convention

feature/nav
feature/hero
feature/product
feature/features
feature/footer


### Workflow

1. Pull the latest version of `main`
2. Create a new branch  
3. Work only on your assigned section
4. Commit changes
5. Push branch to GitHub
6. Open a Pull Request
7. Project Manager reviews and merges into `main`
